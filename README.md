# Sudishwaaran P — 3D Portfolio

A responsive, animated personal portfolio built with Vite, vanilla JavaScript, CSS, and Three.js.

## Run locally

```bash
npm install
npm run dev
```

Open the localhost URL printed by Vite. For a production build:

```bash
npm run build
npm run preview
```

## Customize

- Edit portfolio text and projects in `src/main.js`.
- Edit colors, layout, animation, and responsive styles in `src/style.css`.
- Update email, phone, and LinkedIn URLs in `src/main.js`.

## Stack

Vite · Three.js · CSS animations · Responsive design
